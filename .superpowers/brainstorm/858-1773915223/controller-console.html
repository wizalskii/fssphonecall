One wide panel: position/freq display at top, phone line status area in the middle (shows caller ID, line state with LEDs), PTT bar, and action buttons at bottom. Clean, all info visible at once. Feels like a Zetron dispatch console.

B: Stacked Rack Units

Three stacked "rack-mount" units: position bar at top, large caller display in the middle (big callsign, duration, quality meter), control panel at bottom. Feels like 19" rack equipment. The incoming call gets maximum visual prominence with a large amber display.

Shared Details

  • Amber LCD for controller (ATC standard), vs green for pilot
  • Both have a "Go Online" setup screen before the console appears (callsign + frequency input on dark panel)
  • PTT bar shows red glow when transmitting
  • Go Offline button always accessible
