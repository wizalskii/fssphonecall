B: ATC Desk Console

Professional ATC workstation aesthetic with amber LCD displays, status LEDs, multi-line phone console with line indicators. Wider layout, more information density. Feels like sitting at a real controller position.

C: Hybrid — Cockpit Panel Device

Combines both: a self-contained device with overhead panel header, large central LCD, and bottom button row. Pilot gets the radio feel, controller gets a wider version with more status info. Same visual language for both roles.

Design Notes

  • Green phosphor LCD for pilot (classic avionics), amber LCD for controller (ATC standard)
  • All options use Share Tech Mono font, hardware button gradients, dark backgrounds
  • Screws, bevels, and panel textures add physical realism
  • PTT indicator glows red when transmitting
  • Option C uses one visual system for both roles, reducing code/design duplication
